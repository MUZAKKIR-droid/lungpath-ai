# 🩻 Lung Cancer Detection Website (Python 3.14 Compatible)

A Flask-based AI demo web app that analyzes CT scan images using a simple PyTorch model (placeholder).
Includes risk assessment and doctor directory pages.

### Features
- Image upload + AI analysis
- Risk information & awareness
- Demo doctor contacts

### Run Locally
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
pip install flask
